To open this repo in Obsidian.md you need the following prerequisites:

# Prerequisits
---
1. Obsidian.md download [here](https://obsidian.md/)
2. Git download [here](https://git-scm.com/install/)
# Setup
---
1. Click on the green "<>Code button" and copy the https link
2. Open a terminal in the folder you would like to store the repo
3. Run the command ```git clone <copied-link>
4. Open Obsidian.md 
5. Click open and browse the folder that was created via the git command
